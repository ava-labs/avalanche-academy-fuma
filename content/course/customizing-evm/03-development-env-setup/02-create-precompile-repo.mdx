Throughout this course we will utilize Precompile-EVM to build customized EVMs. So let's head over to the GitHub repository and create a new repository using the template. Make sure to select Include all branches, so you can access branches with the solutions if you need them.

You can also click on this link to create your repository from our template: [Create Repo from Template](https://github.com/new?template_name=precompile-evm&template_owner=ava-labs&visibility=public)

<Callout title="Caution" type="warn">
Please make sure to select the "Include all branches" option! Otherwise, you will not be able to access any of the academy content requried to pursue this course!
</Callout>

![](/course/customizing-evm/1.png)

Don't clone your repository yet. Depending on your setup, you may need to put it in a certain directory.
